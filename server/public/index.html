Server con Node/Express & Typescript e Visual Studio CODE

Istruzioni

Tramite VS CODE

  1. aprire il task runner della compilazione TS 2 JS con CTRL + SHIFT + B o F1 + Build
  2. far partire il server con il debug abilitato con:
    • F5 partenza e step
    • SHIFT + F5 per fermare
    • CTRL + SHIFT + F5 per farlo ripartire
    • F10 per next line
    • F11 per step in

Oppure da terminale:

  • Avviare la transpilazione ed il server con: npm run dev

REST API Endpoints:

  • GET / restituisce questa pagina di istruzioni
  • GET /api/test/:name restituisce un json con il parametro passato
  • GET /api/test?name=NodeJS restituisce un json con il query string passato
  • POST /api/test restituisce un msg con il name passato (usare POSTMAN)
  • GET /api/test/external/users restituisce un json recuperando dati da un api esterna (request con axios )
